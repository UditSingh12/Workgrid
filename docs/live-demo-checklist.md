# Workgrid Live Demo Checklist

## Before sharing the public link

- Confirm `DEMO_MODE=true`
- Verify demo credentials log in successfully
- Verify `/api/health/ready` returns success
- Verify `/api/health/metrics` returns data
- Confirm README screenshots and diagrams render on GitHub
- Confirm GitHub Actions latest workflow is green
- Confirm the dashboard loads with demo analytics and activity

## Before interviews

- Rehearse the 2-minute demo script
- Keep GitHub repo open with README diagrams
- Keep deployed app open in a separate tab
- Be ready to explain tenant isolation, auth, CI, observability, and deployment
